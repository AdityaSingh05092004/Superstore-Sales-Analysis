# Superstore Sales Analysis Dashboard

## Project Overview
This project is an interactive Power BI dashboard built using Superstore Sales data to analyze:

- Sales Performance
- Profit Analysis
- Regional Trends
- Customer Segments
- Forecasting

## Tools Used
- Power BI
- Excel / CSV
- DAX
- Data Visualization

## Dashboard Features
- Sales KPI Tracking
- Region-wise Sales Analysis
- Profit Insights
- Forecasting Analysis
- Interactive Filters
